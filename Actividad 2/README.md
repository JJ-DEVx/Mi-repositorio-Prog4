Este proyecto consiste en el desarrollo de una aplicación de línea de comandos en Python para la gestión de una biblioteca personal. La aplicación permite al usuario registrar, consultar y administrar información sobre libros, incluyendo título, autor, género y estado de lectura.

El sistema utiliza SQLite como motor de base de datos, lo que permite almacenar la información de forma persistente en un archivo local (biblioteca.db). La interacción con la base de datos se realiza mediante la librería sqlite3 de Python, implementando operaciones básicas CRUD (Crear, Leer, Actualizar y Eliminar).

El programa cuenta con un menú interactivo en consola que facilita la navegación entre las distintas funcionalidades, permitiendo al usuario realizar operaciones de manera sencilla y eficiente.

Este proyecto tiene como objetivo principal reforzar los conocimientos sobre el uso de bases de datos relacionales junto con Python, así como la estructuración de programas modulares.



Para ejecutar correctamente este programa, siga los siguientes pasos:



1\. Descargar o clonar el repositorio en su computadora.



2\. Asegurarse de tener Python 3 instalado.

&#x20;  Puede verificarlo con el comando:

&#x20;  python --version



3\. Ubicarse en la carpeta donde se encuentra el archivo del programa (por ejemplo, biblioteca.py) utilizando la terminal o consola.



4\. Ejecutar el programa con el siguiente comando:

&#x20;  python biblioteca.py



5\. Al iniciar, se mostrará un menú interactivo con las siguientes opciones:

&#x20;  - Agregar libro

&#x20;  - Ver libros

&#x20;  - Actualizar libro

&#x20;  - Eliminar libro

&#x20;  - Buscar libros

&#x20;  - Salir



6\. Seleccionar la opción deseada ingresando el número correspondiente y seguir las instrucciones en pantalla.
